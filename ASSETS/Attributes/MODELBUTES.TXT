/* 
** Skeleton [node group] Attributes
**
** DefaultFrontDeathAni			the default death ani played if we are killed from the front
** DefaultBackDeathAni			the default death ani played if we are killed from behind
** DefaultFrontLongRecoilAni	the default long recoil ani played if we are cheezed from the front (humans only)
** DefaultBackLongRecoilAni		the default long recoil ani played if we are cheezed from behind (humans only)
** DefaultHitNode				the default node that is hit if we ever take damage not associated with a specific node
**
** NodeXXName					the name of the node
** NodeXXFlags					the properties on the node . Flags should be the **SUM** of any of the following:
**								
**								 0	--	(no special flags)
**								 1	--	"critical hit" node (head, for example)
**								 2	--	"talk" node (jaw, for example)
**								 4	--  "tire" node (usually rear wheels)
**								 8	--  "turning tire" node (usually front wheels)
**								16	--	"rotor" node (controls helicopter blades)
**								32  --  "eye" node (only one per ai!!)
**								64  --  "wall stick" node (if hit here with spear, try to stick to wall)
**								
** NodeXXFrontDeathAni			the specific death ani played if we are killed in this node from the front
** NodeXXBackDeathAni			the specific death ani played if we are killed in this node from behind
** NodeXXFrontLongRecoilAni		the specific LongRecoil ani played if we are killed in this node from the front (humans only)
** NodeXXBackLongRecoilAni		the specific LongRecoil ani played if we are killed in this node from behind (humans only)
** NodeXXDamageFactor			the factor by which damage received to this node is scaled
** 
** NodeXXParent					the TRUE parent of this node in the hierarchy
**								
** NodeXXRecoilParent			the parent of this node when we walk the parent chain during a recoil
**								
** NodeXXLocation				the body part associated with the node (for hit location tracking)
**								
**								0	--	unknown/unspecified
**								1	--	Head
**								2	--	Torso
**								3	--	Arm
**								4	--	Leg
**								
** NodeXXRadius					the radius around the node used for determining hit location
**
** NodeXXPriority				the priority around the node used for determining hit location
**
*/
